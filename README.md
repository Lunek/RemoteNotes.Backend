## How to run

This project uses [identity server implementation](https://github.com/Lunek/RemoteNotes.Identity).

**&#42;&#42;&#42;Not yet&#42;&#42;&#42;** Implemented Web API functionality can be demonstrated via [client application](https://github.com/Lunek/RemoteNotes.Frontend).

You can simply run this project from console or via your IDE.

## Documentation

To see API documentation just run the project from console:

> 1. Move to **../RemoteNotes.Backend/RemoteNotes.WebApi** folder, open console and write dotnet run  
> 2. Open browser and enter launch url (by default [https://localhost:5001](https://localhost:5001))
> 3. You should now see API documentation generated by Swagger

or from IDE:
> 1. Run project from your IDE by clicking appropriate button
> 2. If browser didn't open automatically, open it manually
> 3. Under specified launch url (by default [https://localhost:5001](https://localhost:5001)) you shold see API documentation
